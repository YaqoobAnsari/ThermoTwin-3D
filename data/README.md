# data/

Datasets live here but are **git-ignored** (only this README + `.gitkeep` are tracked).

| Dir          | Contents                                                              |
|--------------|----------------------------------------------------------------------|
| `raw/`       | Untouched downloads (TUM2TWIN, TBBR, DOE Reference Buildings, scans). |
| `interim/`   | Partially processed / intermediate artefacts.                        |
| `processed/` | Model-ready tensors, latent grids, FEM GT fields.                    |
| `external/`  | Third-party assets not produced by this repo.                        |

See `../docs/datasets.md` for the full catalogue and provenance.
